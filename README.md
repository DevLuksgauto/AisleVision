# AisleVision
Mariloise Bridal | Bridal Consultant Companion App
A modern, mobile-first companion app for high-end bridal consultants and representatives.

This app is designed specifically for Mariloise, a bridal wear representative, to streamline her client management, dress catalog, and appointment workflow. It transforms the traditional bridal consultation process into a seamless, digital experience.
